# 🚀 Misión Galáctica — Fundamentos Python (ALUMNO)

Versión para practicar: retos claros, pistas y checks rápidos.# mision-galactica

## Cómo usar este cuaderno
1. Completa el código en cada reto.
2. Ejecuta el check si existe.
3. Si falla, corrige y vuelve a intentar.

#### Reto 1 — Variables y tipos
Crea estas variables: `capitana` (texto), `energia` (número), `escudo_activo` (booleano).

#### Reto 2 — Condicionales
Muestra `Aterrizaje autorizado` si `energia >= 40` y `escudo >= 50`; si no, denegado.

#### Reto 3 — Bucles
Recorre `modulos` y muestra `Revisando <modulo>...`

#### Reto 4 — Funciones
Completa `puntos_mision` para que devuelva `tareas*10 + bonus`.

#### Reto 5 — Diccionarios
Crea un diccionario `pilota` con `nombre`, `rol`, `nivel` y muestra nombre + rol.

#### Reto 6 (creativo) — Barra de progreso
Asigna un valor a `progreso` (0-100) y visualiza una barra.

#### Reto 7 (humor) — Ruleta galáctica
Usa `random.choice` para escoger un planeta y mostrar un mensaje.

### Meta final
Si completas los 7 retos, ya tienes base para entrar con más seguridad en pandas y limpieza de datos. ✅
